Estrutura Básica

No Visual Studio Code (VSC) nos podemos acessar uma estrutura básica de HTML escrevendo "!".

O VSC já vem programado para abrir a estrutura que contem o arquivo que é o que vamos fazer "HTML" é o tipo do arquivo, o "HEAD" que é o cabeçalho, o "TITLE" que é o o titulo da pagina que vai aparecer na aba do navegador e tambem o "BODY" que será o corpo da pagina aonde o usuario irá ver o site.

Tags

Nas tags temos as tag padrão aonde a maioria das vezes que você for fazer um site, você irá usar elas.

A primeira tag que podemos falar é a tag "i" quando apertamos a tecla "i" no (VSC) ele já abra e fecha a tag automatico, facilitando nossa vida como programador. A tag "i" ela quer dizer "Italico" ou seja ela deixa nosso Texto Italico.
