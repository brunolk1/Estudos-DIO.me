    Estrutura Básica

    No Visual Studio Code (VSC) nos podemos acessar uma estrutura básica de HTML escrevendo "!".

    O VSC já vem programado para abrir a estrutura que contem o arquivo que é o que vamos fazer "HTML" é o tipo do arquivo, o "HEAD" que é o cabeçalho, o "TITLE" que é o o titulo da pagina que vai aparecer na aba do navegador e tambem o "BODY" que será o corpo da pagina aonde o usuario irá ver o site.

    Tags

    Nas tags temos as tag padrão aonde a maioria das vezes que você for fazer um site, você irá usar elas.

    A primeira tag que podemos falar é a tag "i" quando apertamos a tecla "i" no (VSC) ele já abra e fecha a tag automatico, facilitando nossa vida como programador. A tag "i" ela quer dizer "Italico" ou seja ela deixa nosso Texto Italico.

    Proxima tag que podemos colocar é a "STRONG" que é para deixar seu texto Em Negrito.

    Temos também o "INPUT" ele serve para que possamos colocar um "campo" para que possamos digitar dentro dele, e ele tem varios input que possamos colocar, basta colocar "< input type="text" > para que crie um campo de texto, ou "< input type="number" >"

    Atributos das Tags

    Temos diversos atributos que são globais.

    Temos os 3 mais genericos que pode ir em todas as tag.

    temos o "id" Ex.: em uma tag < strong > se colocamos o id < strong id="identificador" > ele cria uma identificação aonde o CSS consegue encontrar e mudar o que for editado somente na quela tag "strong", o "id" só pode ser usado em uma tag só diferente do "class" que vamos falar em seguida.

    O atributo "class" é bem semalhante ao do "id" mas aqui conseguimos colocar em diversas tag EX.: < strong class="formatar-cabeçalho" > < h2 class="formatar-cabeçalho" >, quando se vai formatar tudo aquilo que for do cabeçalho puxamos no CSS apenas o ".formatar-cabeçalho" que todas as tag que estiver com essa classe terá a mesma formatação.

    E temos o atributo "style" que serve para colocar comando CSS direto ao elemento com o codigo < style="color" >< /style >
